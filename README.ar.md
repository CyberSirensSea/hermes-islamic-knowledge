# مهارة المعرفة الإسلامية لهيرميز

مهارة مرجعية شاملة للمعرفة الإسلامية لمساعد [Hermes Agent](https://github.com/NapthaAI/hermes-agent) — قاعدة معرفة للقرآن والحديث والفقه في سطر الأوامر.

## المحتويات

- **القرآن الكريم** — فهرس كامل لسوره الـ114 (الأسماء بالعربية والصينية والإنجليزية، مكان النزول، عدد الآيات، الموضوعات الرئيسية)، مع تكامل مباشر مع Quran.com API v4 لاسترجاع الآيات مع الترجمات الصينية
- **الحديث الشريف** — كتب الحديث الستة بالإضافة إلى المجموعات التكميلية؛ نظام تصنيف الأحاديث (صحيح/حسن/ضعيف)؛ استرجاع عبر CDN؛ منهجية البحث
- **الفقه الإسلامي** — المذاهب السنية الأربعة (حنفي، مالكي، شافعي، حنبلي) مع مقارنات فقهية؛ أصول الفقه؛ القواعد الفقهية؛ الأحكام التكليفية الخمسة
- **التفسير** — أهم كتب التفسير، تصنيف المناهج، أسباب النزول
- **إطار التحقق المتقاطع** — تحقق متعدد المحاور قبل تقديم أي معلومة (قرآن↔قرآن، قرآن↔حديث، حديث↔حديث، مذهب↔مذهب)

## التثبيت

```bash
mkdir -p ~/.hermes/skills/knowledge/islamic-knowledge
cp SKILL.md ~/.hermes/skills/knowledge/islamic-knowledge/
```

سيكتشف Hermes المهارة تلقائيًا. تُفعّل المهارة تلقائيًا عند السؤال عن أي موضوع إسلامي.

## أمثلة على الاستخدام

بعد التثبيت، اسأل Hermes بشكل طبيعي:

- "ما هي الآية 255 من سورة البقرة؟" — يجلب الآية مع الترجمة الصينية عبر Quran.com API
- "ما الأحاديث الواردة في طلب العلم؟" — يبحث في كتب الحديث مع درجة الصحة
- "ما موقف المذاهب الأربعة من الربا؟" — مقارنة فقهية بين المذاهب
- "ما فضل سورة يس؟" — معلومات عن السورة مع الأحاديث ذات الصلة
- "هل يتفق القرآن والسنة في موقفهما من الخمر؟" — تحقق متقاطع بين القرآن والحديث

## المميزات

- لغة الإخراج الافتراضية: الصينية المبسطة
- تكامل مباشر مع Quran.com v4 (بدون مفتاح API)
- نصوص الأحاديث بالإنجليزية عبر CDN + البحث بالصينية عبر الويب
- جداول مقارنة بين المذاهب الفقهية
- حدود صادقة: يُميّز بين آراء العلماء والأحكام الشرعية، ويُشير إلى الخلافات بين المذاهب
- قائمة تحقق قبل تقديم أي معلومة إسلامية

## بيانات المهارة

- **الاسم**: islamic-knowledge
- **التصنيف**: knowledge
- **اللغة الافتراضية**: الصينية المبسطة
- **واجهات API المستخدمة**: Quran.com v4، MuslimSalat.com، Hadith CDN (jsDelivr)
